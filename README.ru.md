## Обзор

ClipDropper — это двухкомпонентное приложение: агент в системном трее Windows и приложение-компаньон для iPhone, которые синхронизируют буфер обмена через локальное Bluetooth-соединение.

- Скопируйте текст или изображение на ПК → мгновенно доступно для вставки на iPhone
- Скопируйте на iPhone → вставляется в Windows
- Отправьте любой файл или папку из проводника Windows правым щелчком мыши
- Всё остаётся локальным — без интернета, без сторонних серверов

| | Функция | Подробности |
|---|---|---|
| **Буфер обмена** | Синхронизация текста | Скопируйте на одном устройстве, вставьте на другом |
| **Буфер обмена** | Синхронизация изображений | Скриншоты и скопированные изображения передаются без проблем |

| **Windows** | Системный трей | Работает в фоновом режиме |
| **Windows** | Автозапуск | Опциональный запуск вместе с Windows |
| **Windows** | Контекстное меню | Интеграция с проводником Windows |
| **История** | Журнал передач | Просмотр всего отправленного |
| **Конфиденциальность** | Только локально | Bluetooth + локальная сеть — без облака |

## Как это работает

ClipDropper использует **Bluetooth Low Energy (BLE)** для обнаружения устройств и передачи малых данных, переключаясь на **локальный HTTP-сервер** для больших передач, таких как файлы и изображения.

```
┌──────────────────────────┐                      ┌──────────────────────────┐
│       Windows PC         │                      │       iPhone (iOS)       │
│                          │                      │                          │
│   ClipDropper.exe        │◄──── BLE GATT ──────►│   Приложение ClipDropper │
│   (Системный трей)       │   (текст, команды)   │   (React Native)         │
│                          │                      │                          │
│   Локальный HTTP-сервер  │◄── Локальная сеть ──►│                          │
│   (токен-аутентификация) │   (файлы, изобр.)    │                          │
└──────────────────────────┘                      └──────────────────────────┘
```

1. Приложение Windows публикует BLE GATT-периферию
2. Приложение iPhone выполняет сканирование и подключается
3. Через BLE обменивается токен аутентификации
4. Текст и небольшие данные передаются через характеристики BLE
5. Файлы и изображения используют HTTP-сервер в локальной сети, защищённый одноразовым токеном

## Установка

### Windows

1. Перейдите на страницу [Releases](https://github.com/emirhan-sonmez/ClipDropper/releases) и скачайте `ClipDropper-Setup.exe`
2. Запустите установщик — .NET 8 Desktop Runtime обнаруживается и устанавливается автоматически при отсутствии
3. Запустите ClipDropper из меню «Пуск» или с ярлыка на рабочем столе

**Альтернатива: запуск без установщика**

Единственная зависимость — бесплатный [SDK .NET 8](https://dotnet.microsoft.com/download/dotnet/8.0) (~200 МБ, однократная установка).

1. Установите [SDK .NET 8](https://dotnet.microsoft.com/download/dotnet/8.0)
2. Скачайте или клонируйте этот репозиторий
3. Дважды щёлкните `ClipDropper-Windows\run.bat` — приложение соберётся и запустится автоматически

### iOS

Приложение iOS ещё не в App Store. Вы можете бесплатно установить его на свой iPhone с помощью **Sideloadly** — без аккаунта разработчика и без джейлбрейка.

> **Внимание:** Приложения, загруженные с бесплатного Apple ID, истекают через **7 дней** и требуют повторной подписи. Sideloadly может делать это автоматически, когда телефон подключён.

#### Шаг 1 — Скачайте файлы

- Скачайте **Sideloadly** (бесплатно): [sideloadly.io](https://sideloadly.io)
- Скачайте `ClipDropper.ipa` со страницы [Releases](https://github.com/emirhan-sonmez/ClipDropper/releases)

#### Шаг 2 — Установите на iPhone

1. Подключите iPhone к ПК через USB
2. Если iPhone запросит, нажмите **Доверять этому компьютеру**
3. Откройте Sideloadly и перетащите `ClipDropper.ipa` в окно
4. Введите Apple ID и нажмите **Start**
5. Дождитесь завершения установки

#### Шаг 3 — Доверяйте приложению на iPhone

1. Перейдите в **Настройки → Основные → VPN и управление устройством**
2. В разделе **Приложение разработчика** найдите ваш Apple ID
3. Нажмите **Доверять "[ваш Apple ID]"** → **Доверять**

#### Шаг 4 — Включите режим разработчика (iOS 16 и новее)

1. Перейдите в **Настройки → Конфиденциальность и безопасность → Режим разработчика**
2. Включите его
3. Нажмите **Перезапустить** при запросе
4. После перезапуска нажмите **Включить** для подтверждения

## Сборка из исходников

### Требования

| Инструмент | Минимальная версия |
|------|----------------|
| .NET SDK | 8.0 |
| Windows | 10 (build 19041+, x64) |
| Node.js | 18+ |
| Inno Setup | 6.x _(только для установщика)_ |

### Приложение Windows

```sh
cd ClipDropper-Windows
dotnet run
```

### Установщик Windows

```sh
ClipDropper-Windows\build-installer.bat
```

### Приложение iOS

```sh
cd ClipDropper-iOS
npm install
npx expo start
```
